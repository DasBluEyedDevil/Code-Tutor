## ðŸ”§ Troubleshooting

### "Code execution failed" error?
**Cause**: The language runtime isn't installed

**Fix**:
1. Install the language runtime (see "Want to Execute Code?" above)
2. Restart Code Tutor
3. Try running code again

### Can't find the app after installing?
**Fix**: Search for "Code Tutor" in the Windows Start Menu

### My progress disappeared!
**Cause**: Progress is saved per Windows user account

**Fix**: Make sure you're logged into the same Windows account

### The app won't start
**Fix**:
1. Check Windows Event Viewer for errors
2. Try reinstalling: Uninstall â†’ Download new installer â†’ Install
3. Report the issue on GitHub: https://github.com/DasBluEyedDevil/Code-Tutor/issues
