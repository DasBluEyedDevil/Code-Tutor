Excel spreadsheet:
| ID | Name          | Age | Email              | GPA  |
|----|---------------|-----|--------------------| -----|
| 1  | Alice Johnson | 20  | alice@example.com  | 3.75 |
| 2  | Bob Smith     | 21  | bob@example.com    | 3.50 |
| 3  | Carol Davis   | 19  | carol@example.com  | 3.90 |

Database table = Same thing!
- Each ROW = One student
- Each COLUMN = One property
- Each CELL = One value

SQL just lets you manipulate this "spreadsheet" with commands.
