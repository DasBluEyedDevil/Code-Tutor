title: "The Problem: Java Needs to Talk to Databases"
---

So far:
- You know Java (objects, methods, classes)
- You know SQL (SELECT, INSERT, JOIN)

But they're separate!

How do you run SQL from Java code?
How do you get query results into Java objects?

Solution: JDBC (Java Database Connectivity)
- Standard Java API for database access
- Works with MySQL, PostgreSQL, Oracle, etc.
- Lets you execute SQL from Java code
