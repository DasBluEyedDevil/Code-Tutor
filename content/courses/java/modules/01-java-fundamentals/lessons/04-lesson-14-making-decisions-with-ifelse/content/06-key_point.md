✓ 'if' statements let your program make decisions
✓ The condition in parentheses must be true or false
✓ Only the code in the TRUE block runs
✓ Use 'else' to handle the FALSE case
✓ Use == to compare (not =)

Think of it as:
if (something is true) {
    do this
} else {
    do that instead
}
