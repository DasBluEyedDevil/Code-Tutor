type: "KEY_POINT"
title: "The Web is Like a Restaurant"
---

YOU (Client):
- Sit at a table
- Look at menu
- Order food: "I'll have the pasta"

KITCHEN (Server):
- Receives your order
- Prepares the food
- Sends it back to you

WAITER (HTTP Protocol):
- Carries messages between you and kitchen
- Speaks a specific language (HTTP)

In web terms:
- Your browser = Client (customer)
- Java server = Server (kitchen)
- HTTP = Protocol (waiter's language)
