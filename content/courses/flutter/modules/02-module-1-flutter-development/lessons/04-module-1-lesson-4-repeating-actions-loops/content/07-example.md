### Example 3: Sum of Numbers




```dart
void main() {
  var sum = 0;

  for (var i = 1; i <= 10; i++) {
    sum += i;  // Same as: sum = sum + i
  }

  print('Sum of 1 to 10 is: $sum');  // Output: 55
}
```
