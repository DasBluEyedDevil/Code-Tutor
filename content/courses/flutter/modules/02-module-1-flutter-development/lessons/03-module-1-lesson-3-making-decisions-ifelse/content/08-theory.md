title: "Combining Conditions: AND / OR"
---


Sometimes you need to check multiple things at once.

### AND (&&) - Both must be true


### OR (||) - At least one must be true


### NOT (!) - Flips true/false




```dart
void main() {
  var isRaining = false;

  if (!isRaining) {
    print('It\'s not raining. Let\'s go outside!');
  }
}
```
