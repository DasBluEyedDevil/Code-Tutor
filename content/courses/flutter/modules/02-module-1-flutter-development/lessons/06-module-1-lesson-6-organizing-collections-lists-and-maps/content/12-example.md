title: "Creating Your First Map"
---



**The Parts**:
- **`{}`** - Curly braces indicate a Map
- **`'Alice':`** - The key
- **`'555-1234'`** - The value
- **`,`** - Separates pairs



```dart
void main() {
  var phoneBook = {
    'Alice': '555-1234',
    'Bob': '555-5678',
    'Carol': '555-9012'
  };

  print(phoneBook);
}
```
