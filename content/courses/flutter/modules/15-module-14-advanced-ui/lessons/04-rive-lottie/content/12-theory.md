---


**Decision Framework:**

| Requirement | Choose Lottie | Choose Rive |
|-------------|--------------|-------------|
| Team has After Effects experience | Yes | - |
| Need state machine logic | - | Yes |
| Simple playback animations | Yes | - |
| Interactive/responsive animations | - | Yes |
| Large animation library available | Yes | - |
| Complex character animations | - | Yes |
| Game-like UI elements | - | Yes |
| Quick prototyping | Yes | - |

**Practical Recommendations:**

**Use Lottie for:**
- Loading spinners and progress indicators
- Success/error/empty state illustrations
- Animated icons and micro-interactions
- Onboarding illustrations
- Decorative background animations

**Use Rive for:**
- Interactive buttons and toggles
- Character animations (avatars, mascots)
- Data-driven visualizations
- Animated form feedback
- Game elements and gamification

**Can You Use Both?**

Absolutely! Many apps use both:
- Lottie for simple decorative animations
- Rive for interactive elements

Choose based on the specific needs of each animation.
