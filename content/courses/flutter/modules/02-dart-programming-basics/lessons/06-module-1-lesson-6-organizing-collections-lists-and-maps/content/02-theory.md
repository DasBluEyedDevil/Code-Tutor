title: "What is a List?"
---


**Conceptual Explanation**:
A List is like a numbered container with multiple compartments, each holding one item.


**Note**: Lists start counting at **0**, not 1! This is called "zero-indexing."



```dart
List of fruits:
[0] Apple
[1] Banana
[2] Orange
[3] Mango
```
