**The Parts**:

1. **`void`** - This means "doesn't give anything back" (we'll learn about returning values soon)
2. **`sayHello`** - The function name (use camelCase)
3. **`()`** - Parameters go here (empty for now)
4. **`{ }`** - The function body (code to run)



```dart
void sayHello() {
  print('Hello!');
}
```
