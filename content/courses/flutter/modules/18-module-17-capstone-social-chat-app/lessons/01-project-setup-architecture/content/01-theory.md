**The Social Chat App Capstone**

Welcome to the capstone project! Over the next several lessons, you'll build a complete social chat application from scratch. This isn't a toy project - it's a production-ready app that combines everything you've learned throughout this course.

**Feature Overview**

Our social chat app will include:

1. **Authentication & Profiles**
   - Email/password registration and login
   - User profiles with avatars
   - Account settings and preferences

2. **Real-time Messaging**
   - One-on-one direct messages
   - Group chat rooms
   - Typing indicators
   - Read receipts
   - Message history with pagination

3. **Social Features**
   - Friend requests and connections
   - User search and discovery
   - Online/offline status
   - Block and report functionality

4. **Media & Content**
   - Image sharing in chats
   - Link previews
   - Emoji reactions
   - Message editing and deletion

5. **Push Notifications**
   - New message alerts
   - Friend request notifications
   - Background message sync

**Why a Chat App?**

A chat application is the perfect capstone because it touches every major concept:

| Concept | Application |
|---------|-------------|
| State Management | Real-time message updates, typing indicators |
| Backend APIs | REST endpoints for users, messages, rooms |
| WebSockets | Live message delivery, presence updates |
| Database Design | Users, messages, rooms, relationships |
| Authentication | Secure login, token management |
| File Storage | Avatar uploads, image sharing |
| Push Notifications | Message alerts |
| UI/UX | Chat bubbles, animations, gestures |
