title: "Setting Up Testing Dependencies"
---


**Add to `pubspec.yaml`:**
```yaml
dev_dependencies:
  flutter_test:
    sdk: flutter
  mocktail: ^1.0.0
```

**Mocktail** is the modern mocking library for Dart:
- No code generation required
- Clean, readable syntax
- Works with null safety
