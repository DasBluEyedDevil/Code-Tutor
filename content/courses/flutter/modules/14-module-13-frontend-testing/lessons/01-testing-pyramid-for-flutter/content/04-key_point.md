type: "KEY_POINT"
title: "When to Use Each Test Type"
---


| Test Type | Use For | Speed | Confidence |
|-----------|---------|-------|------------|
| Unit | Business logic, utilities | Fast | Lower |
| Widget | UI components, interactions | Medium | Medium |
| Integration | User journeys, E2E | Slow | Highest |

**Rule of Thumb:** 70% unit, 20% widget, 10% integration.
