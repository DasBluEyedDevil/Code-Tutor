title: "Firebase Project Structure"
---


After setup, your Firebase project has:

### 1. **Console** (https://console.firebase.google.com)
- View data
- Manage users
- Monitor usage
- Configure settings

### 2. **Authentication**
- User management
- Sign-in methods configuration

### 3. **Firestore Database**
- NoSQL database
- Collections and documents
- Security rules

### 4. **Storage**
- File uploads
- Access control

### 5. **Settings**
- API keys
- Project settings
- Team members
