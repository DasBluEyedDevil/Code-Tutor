title: "Installing the VS Code Extension"
---


Serverpod has an official VS Code extension that improves the development experience significantly.

**Installation:**

1. Open VS Code
2. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS)
3. Search for "Serverpod"
4. Install the extension by Serverpod (the official one)

**Features the extension provides:**

- **Syntax highlighting** for protocol YAML files
- **Code snippets** for common patterns
- **Quick fixes** and suggestions
- **Generate command** integration
- **Error detection** in protocol definitions

**Recommended additional extensions for Dart backend development:**

- **Dart** (by Dart Code) - Essential for any Dart development
- **Docker** (by Microsoft) - Manage containers from VS Code
- **YAML** (by Red Hat) - Better YAML editing
- **PostgreSQL** (by Chris Kolkman) - Database integration
