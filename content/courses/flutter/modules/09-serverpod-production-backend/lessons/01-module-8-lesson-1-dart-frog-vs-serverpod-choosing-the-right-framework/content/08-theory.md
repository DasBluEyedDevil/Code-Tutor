title: "Feature-by-Feature Comparison"
---


| Feature | Dart Frog | Serverpod |
|---------|-----------|----------|
| **Routing** | File-based, automatic | Endpoint classes, code-gen |
| **Database** | Bring your own | Built-in ORM (PostgreSQL) |
| **Authentication** | Bring your own | Built-in (email, OAuth, etc.) |
| **Real-time** | Manual WebSocket | Built-in streaming |
| **File Storage** | Manual | Built-in (S3-compatible) |
| **Caching** | Manual | Built-in (Redis) |
| **Client Code** | Manual | Auto-generated |
| **Type Safety** | Partial | Full end-to-end |
| **Learning Curve** | Lower | Higher |
| **Setup Time** | Minutes | 15-30 minutes |
| **Dependencies** | Minimal | Docker, PostgreSQL |
| **Deployment** | Any Dart host | Needs PostgreSQL |
| **Best For** | Prototypes, simple APIs | Production apps |
