**App Store Product Page Elements:**

**App Name (30 characters):**
- Clear, memorable, searchable
- Include key functionality if space allows
- Avoid generic terms unless descriptive

**Subtitle (30 characters):**
- Summarize app's purpose
- Include secondary keywords
- Complement, don't repeat, the name

**Keywords (100 characters):**
- Comma-separated, no spaces after commas
- Include synonyms and related terms
- Don't repeat words from name or subtitle
- Research competitor keywords
- Update keywords with each version

**Description (4,000 characters):**
- First 1-3 sentences visible without expanding
- Lead with key benefits
- Use line breaks for readability
- Include features with bullet points
- Add social proof (awards, reviews)
- End with call to action
- No HTML formatting supported

**Screenshots:**
- iPhone: 6.7", 6.5", 5.5" sizes
- iPad: 12.9" required if app runs on iPad
- Up to 10 screenshots per device
- First 3 most important (visible in search)
- Show real app UI with context
- Add captions for feature explanation

**App Preview (Video):**
- 15-30 seconds recommended
- Captures real device usage
- No external footage or actors
- First frame used as poster image
- Approved devices only for recording
