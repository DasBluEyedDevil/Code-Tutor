**Before You Can Submit:**

1. **Signed Release APK or App Bundle:**
   - App bundle (.aab) preferred - Google generates optimized APKs
   - Or signed APK (.apk) for legacy compatibility
   - Must be signed with your upload key (or app signing key if not using Play App Signing)

2. **App Content:**
   - All features must work without crashes
   - No placeholder content or "lorem ipsum" text
   - All links must work and lead to valid destinations

3. **Privacy Policy:**
   - Required if app collects personal data
   - Must be hosted on a publicly accessible URL
   - Required for apps targeting children or using sensitive permissions

4. **Target API Level:**
   - New apps must target recent Android API level
   - Currently API 34 (Android 14) for new apps
   - Updates to existing apps have slightly lower requirements

5. **Store Listing Assets:**
   - App icon, screenshots, feature graphic
   - Short and full descriptions
   - Category and contact information
