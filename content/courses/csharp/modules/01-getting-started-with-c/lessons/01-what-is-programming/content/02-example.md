This example shows your first C# program using Console.WriteLine to display text to the screen.

```csharp
// YOUR FIRST C# PROGRAM!
// This is a comment - the computer ignores it
// Comments help humans understand the code

Console.WriteLine("Hello, World!");

// Let's break this down:
// Console = The black window where text appears
// WriteLine = Write a line of text
// "Hello, World!" = The text to display
// ; = End of instruction (like a period in English)

// More examples:
Console.WriteLine("Welcome to C#!");
Console.WriteLine("Programming is fun!");
Console.WriteLine("I am learning C#!");

// Each instruction runs in order:
Console.WriteLine("First");   // Runs 1st
Console.WriteLine("Second");  // Runs 2nd
Console.WriteLine("Third");   // Runs 3rd

// Output:
// First
// Second
// Third
```
