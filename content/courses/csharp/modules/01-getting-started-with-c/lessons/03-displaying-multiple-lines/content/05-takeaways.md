type: "KEY_POINT"
title: "Key Takeaways"
---

- **`\n`** is the escape sequence for a new line.
- **Escape Sequences** start with a backslash (`\`), not a forward slash (`/`).
- **`Console.Write()`** prints text without starting a new line.
- **`Console.WriteLine()`** prints text and then starts a new line.
- **`\t`** adds a tab (indentation).
- **`\"`** allows you to put double quotes inside a string.
