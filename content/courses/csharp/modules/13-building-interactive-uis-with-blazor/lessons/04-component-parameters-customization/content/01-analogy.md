Parameters are like function arguments for components!

Function with parameters:
void Greet(string name, int age) { }

Component with parameters:
<Person Name="Alice" Age="30" />

Parameters let you:
✅ Customize component behavior
✅ Pass data from parent to child
✅ Make components reusable
✅ Configure appearance/logic

Types of parameters:
• Simple values (string, int, bool)
• Complex objects (Product, Customer)
• Collections (List<T>, arrays)
• Event callbacks (EventCallback<T>)
• RenderFragments (child content)

Think: Parameters = 'The inputs that make your component flexible and reusable!'
