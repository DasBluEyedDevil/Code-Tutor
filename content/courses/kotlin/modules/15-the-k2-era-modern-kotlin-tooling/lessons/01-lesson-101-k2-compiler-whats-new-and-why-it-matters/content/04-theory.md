### Up to 2x Faster Compilation

K2 delivers significant performance improvements across the board:

**Compilation Speed**
- Clean builds: 1.5-2x faster
- Incremental builds: Up to 2x faster
- IDE analysis: Noticeably faster

**Memory Usage**
- Lower memory footprint during compilation
- Better garbage collection behavior
- Reduced peak memory usage

**Real-World Results (JetBrains Data)**

| Project | K1 Clean Build | K2 Clean Build | Improvement |
|---------|---------------|----------------|-------------|
| Kotlin Compiler | 54s | 29s | 1.9x |
| IntelliJ IDEA | 8m 12s | 4m 45s | 1.7x |
| YouTrack | 2m 15s | 1m 20s | 1.7x |
