### What You Need

1. **Apple Developer Account** ($99/year)
   - Enroll at [developer.apple.com/enroll](https://developer.apple.com/enroll)

2. **App Store Connect Account**
   - Automatically included with developer account
   - Access at [appstoreconnect.apple.com](https://appstoreconnect.apple.com)

3. **Certificates and Provisioning Profiles**
   - Distribution certificate
   - App Store provisioning profile

4. **App Store Assets**
   - App icon (1024x1024 PNG, no alpha)
   - Screenshots for each device size
   - App description, keywords
   - Privacy policy URL
   - Support URL

5. **Xcode Archive**
   - Built with Release configuration
   - Signed with distribution certificate
