### Question 1: K2 Compiler

What is the main benefit of the K2 compiler?

**A)** New syntax features
**B)** Faster compilation and better IDE performance
**C)** Incompatible with old code
**D)** Only works on Linux

**Answer**: **B** - The K2 compiler provides faster compilation (up to 2x) and significantly improved IDE performance.

---

### Question 2: Backward Compatibility

Does existing Kotlin code work with the Kotlin 2.x series?

**A)** No, all code must be rewritten
**B)** Only code written after 2020
**C)** Yes, most existing code works unchanged
**D)** Only with special migration tools

**Answer**: **C** - The Kotlin 2.x series maintains backward compatibility, and most existing code works without changes.

---

### Question 3: Smart Casts

What improvement does K2 bring to smart casts?

**A)** Smart casts are removed
**B)** Smart casts work in more scenarios and across more conditions
**C)** Smart casts now require explicit syntax
**D)** No changes to smart casts

**Answer**: **B** - K2 improves smart casts to work in more scenarios, including complex conditions and property accesses.
