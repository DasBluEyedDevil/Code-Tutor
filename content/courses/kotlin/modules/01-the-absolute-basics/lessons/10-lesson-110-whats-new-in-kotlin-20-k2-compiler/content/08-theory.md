### Performance Improvements

| Improvement | Benefit |
|-------------|--------|
| K2 Compiler | Up to 2x faster compilation |
| IDE Support | Faster code completion and analysis |
| Memory Usage | More efficient compiler memory usage |
| Native Builds | Faster Kotlin/Native compilation |

### Language Improvements (2.0 through 2.3)

| Feature | Version | Description |
|---------|---------|-------------|
| Smart Casts | 2.0 | Work in more scenarios |
| Type Inference | 2.0 | Better without explicit types |
| Guard Conditions | 2.2 | `when` branches with `if` guards |
| Non-local break/continue | 2.2 | Break/continue in lambdas |
| Multi-dollar interpolation | 2.2 | `$$` for literal dollar signs in strings |
| Context Parameters | 2.2+ | Implicit parameter passing (preview) |
| Explicit Backing Fields | 2.3 | Custom backing field types for properties |
| Java 25 Support | 2.3 | Latest JDK interop |

### What Stays the Same

- Basic syntax and language features
- Standard library functions
- Interoperability with Java
- Coroutines and Flow APIs
