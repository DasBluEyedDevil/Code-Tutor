**Goal**: Create a calculator that adds two numbers.

**Requirements**:
1. Ask for first number
2. Ask for second number
3. Add them together
4. Print the result

**Hint**: Use `readln().toInt()` to read numbers.

**Starter Code**:

```kotlin
fun main() {
    println("=== Simple Calculator ===")
    // Your code here
}
```

**Expected Output**:

```text
=== Simple Calculator ===
Enter first number:
10
Enter second number:
5
10 + 5 = 15
```
