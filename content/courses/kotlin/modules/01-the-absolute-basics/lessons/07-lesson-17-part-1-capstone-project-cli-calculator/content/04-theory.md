We'll structure our calculator with these components:


---



```kotlin
1. Data Models
   - Calculation (stores a single calculation)

2. Core Functions
   - add(), subtract(), multiply(), divide(), modulus()
   - formatResult()

3. UI Functions
   - displayMenu()
   - displayHistory()
   - clearHistory()

4. Input Functions
   - getNumber()
   - getMenuChoice()

5. Main Program
   - Main loop
   - Menu handling
   - Operation execution
```
