**Estimated Time**: 60 minutes

Welcome to Lesson 1.4! As your programs grow, you'll find yourself wanting to reuse code rather than copying and pasting it. This is where **Functions** come in.

In this lesson, you will learn:
1. **Function Basics**: How to define and call your own reusable blocks of code.
2. **Parameters**: How to pass data into functions to make them flexible.
3. **Return Values**: How to get results back from a function.
4. **Single-Expression Functions**: A concise Kotlin-specific way to write simple functions.
5. **Default and Named Arguments**: How to make function calls more readable and flexible.

By the end of this lesson, you'll be able to organize your code like a professional, making it modular, readable, and easy to maintain!
