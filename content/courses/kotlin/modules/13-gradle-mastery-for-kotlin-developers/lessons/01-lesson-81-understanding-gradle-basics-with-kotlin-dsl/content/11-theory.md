---


**Goal**: Set up a new Kotlin project from scratch.

**Steps**:
1. Create project directory
2. Initialize Gradle wrapper
3. Create settings.gradle.kts
4. Create build.gradle.kts with Kotlin JVM plugin
5. Create a simple Main.kt
6. Run with `./gradlew run`
