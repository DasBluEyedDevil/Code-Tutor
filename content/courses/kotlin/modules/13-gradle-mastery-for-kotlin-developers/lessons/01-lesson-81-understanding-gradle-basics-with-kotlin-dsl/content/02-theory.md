Gradle is the build system that powers Kotlin projects, Android apps, and modern JVM development. Understanding Gradle is essential for professional Kotlin development.

In this lesson, you'll learn:
- Gradle's task-based build model
- How to read and write `build.gradle.kts` files
- The difference between project and build scripts
- How to configure basic project metadata
