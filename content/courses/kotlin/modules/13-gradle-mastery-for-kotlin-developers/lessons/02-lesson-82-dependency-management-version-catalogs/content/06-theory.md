### Why Use Version Catalogs?

**1. Single Source of Truth**
- All versions in one file
- No version mismatches across modules

**2. IDE Support**
- Autocomplete for dependencies
- Type-safe references
- Refactoring support

**3. Easy Updates**
- Update version once, applies everywhere
- Clear view of all dependencies

**4. Bundles**
- Group related dependencies
- Add multiple libraries with one line
