title: "Exercise: Set Up Version Catalog"
---


**Goal**: Create a version catalog for a Ktor server project.

**Requirements**:
1. Create `gradle/libs.versions.toml`
2. Define versions for Kotlin, Ktor, and kotlinx.serialization
3. Create library entries for ktor-server-core, ktor-server-netty, ktor-serialization
4. Create a bundle for all Ktor server dependencies
5. Update build.gradle.kts to use the catalog
