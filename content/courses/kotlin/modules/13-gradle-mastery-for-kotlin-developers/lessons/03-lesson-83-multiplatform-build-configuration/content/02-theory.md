Kotlin Multiplatform (KMP) projects require special Gradle configuration to target multiple platforms. Understanding how to configure targets, source sets, and platform-specific dependencies is crucial.

In this lesson, you'll learn:
- How to configure Kotlin Multiplatform targets
- Setting up shared source sets
- Handling platform-specific dependencies
- Configuring iOS framework distribution
