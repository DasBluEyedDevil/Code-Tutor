title: "Exercise: Create Team Conventions"
---


**Goal**: Create a set of convention plugins for a team.

**Requirements**:
1. Create buildSrc with kotlin-dsl
2. Create `kotlin-base-conventions` with JVM 17 and warnings-as-errors
3. Create `kotlin-library-conventions` that applies base + explicit API mode
4. Create `kotlin-app-conventions` that applies base + application plugin
5. Apply to sample modules
