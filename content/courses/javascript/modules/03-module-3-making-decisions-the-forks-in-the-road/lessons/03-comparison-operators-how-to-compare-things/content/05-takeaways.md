type: "TAKEAWAY"
title: "Key Takeaways"
---

*   **Result**: All comparisons return a Boolean (`true` / `false`).
*   **Strict Equality**: Use `===` to check if values are identical.
*   **Strict Inequality**: Use `!==` to check if values are different.
*   **Math Comparisons**: Use `>`, `<`, `>=`, `<=` for numbers.
*   **Assignment vs Comparison**: Never confuse `=` (set value) with `===` (check value).
