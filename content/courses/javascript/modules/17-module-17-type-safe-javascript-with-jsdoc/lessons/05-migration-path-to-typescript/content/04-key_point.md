title: "Bun Makes This Easy"
---

With Bun, there's no compilation step for TypeScript:

```bash
# JavaScript with JSDoc
bun run index.js

# TypeScript
bun run index.ts

# Both work the same way - no tsc, no build!
```

You can even mix `.js` and `.ts` files in the same project during migration.
