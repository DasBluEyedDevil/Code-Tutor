title: "Key Takeaways"
---

*   **Strings**: Text data, wrapped in quotes (`'...'`, `"..."`) or backticks (`` `...` ``).
*   **Template Literals**: Use backticks and `${var}` to embed variables in strings.
*   **Numbers**: Can be integers (`10`) or decimals (`10.5`).
*   **Booleans**: `true` or `false` (logic switches).
*   **`undefined`**: Variable exists but has no value (default).
*   **`null`**: Variable is intentionally empty.
*   **`typeof`**: Checks the type of a value.
