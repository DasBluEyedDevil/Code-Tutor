**Modern React Project Setup:**\n\n```bash\n# Create new React project with Vite (RECOMMENDED)\nnpm create vite@latest my-app -- --template react\ncd my-app\nnpm install\nnpm run dev\n```\n\n**Why Vite over Create React App?**\n- Create React App (CRA) is **deprecated** as of 2023\n- Vite is 10-100x faster for development\n- Native ES modules support\n- Recommended by React documentation\n\n**Alternative: Next.js** (for full-stack React):\n```bash\nnpx create-next-app@latest my-app\n```\n\nAlways use Vite or Next.js for new React projects in 2025!
