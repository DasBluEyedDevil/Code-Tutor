title: "Key Takeaways"
---

*   **Programming** is giving clear, step-by-step instructions to a computer.
*   **JavaScript** is the language we are using (primarily for the web).
*   **`console.log()`** is the command to print output to the screen.
*   **Statements** usually end with a semicolon (`;`).
*   **Case Sensitivity** matters: `console.log` is not the same as `Console.log`.
