type: "TAKEAWAY"
title: "Key Takeaways"
---

*   Code runs from **top to bottom**.
*   **Math** follows standard order of operations (Multiplication before Addition).
*   Use **Template Literals** (backticks `` `...` `` and `${...}`) to combine text and values.
*   **Strings** are for text, **Numbers** are for math.
