---
type: "TAKEAWAY"
title: "Key Takeaways"
---

*   **Comments** are ignored by the computer but are vital for humans.
*   Use `//` for single-line comments.
*   Use `/* ... */` for multi-line comments.
*   Use comments to explain **Why** you wrote the code, not just **What** it does.
*   You can "comment out" code to temporarily disable it without deleting it.
