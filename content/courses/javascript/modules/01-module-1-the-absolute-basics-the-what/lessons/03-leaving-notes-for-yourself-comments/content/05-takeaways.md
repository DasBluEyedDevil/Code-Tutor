type: "TAKEAWAY"
title: "Key Takeaways"
---

*   **Comments** are notes for humans that the computer ignores.
*   **`//`** starts a single-line comment.
*   **`/* ... */`** creates a multi-line comment.
*   **Commenting Out** is a technique to temporarily disable code for testing.
*   **Best Practice**: Explain **Why**, not **What**.
