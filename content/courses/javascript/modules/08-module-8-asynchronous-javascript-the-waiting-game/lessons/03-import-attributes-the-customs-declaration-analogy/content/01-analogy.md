title: "The Customs Declaration"
---

Imagine you're receiving packages at a shipping dock. Some packages contain food, others contain electronics, and some might even contain hazardous materials. You wouldn't want to handle them all the same way!

Import Attributes work like customs declarations. When you import a file, you tell JavaScript EXACTLY what type of content you're expecting. This prevents security issues where a .json file could secretly contain executable code.
