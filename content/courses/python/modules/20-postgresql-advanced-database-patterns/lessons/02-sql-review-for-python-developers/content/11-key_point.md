title: "Key Takeaways"
---

- **RETURNING** clause gets inserted/updated rows immediately
- **JOINs** combine related tables - use INNER for matching, LEFT for all from one side
- **GROUP BY + HAVING** aggregates data and filters groups
- **CTEs** (WITH clause) break complex queries into readable steps
- **Window functions** calculate across rows without collapsing them
- **LAG/LEAD** access previous/next rows for trend analysis
- **ROW_NUMBER/RANK** for ordering and pagination
- **Running totals** use `SUM() OVER (ORDER BY ...)`
