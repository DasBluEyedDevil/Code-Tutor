title: "Key Takeaways"
---

- **Lambda syntax**: `lambda parameters: expression` - returns the expression automatically
- **Lambdas are anonymous** - No name needed, but you can assign them to variables
- **Use lambdas for simple operations** - If it needs multiple lines, use a regular function
- **`sorted(key=...)`** - Use lambda to customize sorting
- **`map(func, list)`** - Apply function to every item, returns iterator
- **`filter(func, list)`** - Keep items where function returns True
- **Convert to list**: `map()` and `filter()` return iterators - use `list()` to see results
- **Built-in math functions**: `abs()`, `round()`, `min()`, `max()`, `sum()`
- **Type conversions**: `int()`, `float()`, `str()`, `list()`, `bool()`
- **Lambda + built-ins = powerful one-liners** for data transformation
