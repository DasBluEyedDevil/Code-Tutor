**Expected Output:**
```
Hello! What's your name?
[User types: Sarah]
Nice to meet you, Sarah !

What's your favorite color?
[User types: blue]
Wow! blue is a great color!
```

```python
# Let's create our first interactive program!

# Output: Ask the user for their name
print("Hello! What's your name?")

# Input: Wait for the user to type their name and press Enter
name = input()

# Output: Greet them personally
print("Nice to meet you,", name, "!")

# Let's ask another question
print("\nWhat's your favorite color?")
color = input()

# Respond
print("Wow!", color, "is a great color!")
```
