title: "Key Takeaways"
---

- **ORM maps Python classes to database tables** - Work with objects, not SQL strings
- **SQLAlchemy is the industry standard** - Powers Flask, FastAPI, and many production apps
- **Define models with Column types** - Integer, String, Boolean, DateTime, etc.
- **CRUD is intuitive** - add(), query(), commit(), delete()
- **Relationships link tables** - Use ForeignKey and relationship()
- **Queries are chainable** - filter().order_by().limit().all()
- **Sessions manage transactions** - commit() saves, rollback() reverts
- **Migrations track schema changes** - Use Alembic for production
