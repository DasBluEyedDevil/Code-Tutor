**Git = Save points for your code**

**Think of a video game:**
- Save your progress before boss fight
- Try different strategies
- Go back if you fail
- Keep multiple save files

**Git does this for code!**

**Basic workflow:**
```
1. Clone repo (copy to your computer)
2. Create branch (new feature)
3. Make changes
4. Commit changes (save point)
5. Push to GitHub (backup)
6. Create Pull Request
7. Review + Merge
```

**Why use Git:**
- Track every change ever made
- Collaborate without overwriting others
- Experiment safely
- Revert mistakes
- Required for professional development
