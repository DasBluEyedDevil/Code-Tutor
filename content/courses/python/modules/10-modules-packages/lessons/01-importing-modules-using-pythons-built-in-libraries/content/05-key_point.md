- **Modules = reusable code libraries.** Python has 200+ built-in modules.
- **import module** - Use as module.function()
- **from module import item** - Use item directly
- **import module as alias** - Shorter name (import pandas as pd)
- **Avoid from module import *** - Pollutes namespace
- **Common modules:** math, random, datetime, json, csv, pathlib, os, sys
