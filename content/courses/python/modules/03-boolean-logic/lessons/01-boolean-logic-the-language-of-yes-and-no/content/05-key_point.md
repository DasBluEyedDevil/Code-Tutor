- **Boolean values** are `True` or `False` (capitalized!) - the foundation of all program decisions
- **Comparison operators** ask questions and return Boolean values:
<li>`==` equal, `!=` not equal
- `&gt;` greater, `&lt;` less
- `&gt;=` greater or equal, `&lt;=` less or equal

</li>- **Don't confuse:**
<li>`=` (assignment: "make it equal")
- `==` (comparison: "check if equal")

</li>- **Store Boolean results** in variables with descriptive names for readable code
- **String comparisons** are case-sensitive: "Hello" â‰  "hello"
- **The data type** is called `bool` (check with `type()`)
- **Mathematical truth:** `True == 1` and `False == 0` (useful for counting!)
